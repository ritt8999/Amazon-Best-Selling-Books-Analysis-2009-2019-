#  Amazon Best-Selling Books Analysis (2009–2019)

This project analyzes Amazon’s bestselling books from 2009–2019 using a public Kaggle dataset. The goal is to understand trends in **genres, authors, ratings, reviews, and pricing**, supported by an **interactive Power BI dashboard**.

##  Dataset

Includes: Name, Author, User Rating, Reviews, Price, Year, Genre.

##  Objectives

* Identify trends across years
* Compare Fiction vs Non-Fiction
* Analyze top-performing authors
* Understand pricing and rating patterns
* Build a Power BI dashboard

##  TechTools Used

Python • Pandas • Matplotlib • Power BI

##  Project Steps

1. Load and inspect the dataset
2. Clean and prepare data
3. Perform EDA
4. Visualize findings
5. Build Power BI dashboard


##  Dashboard Name

**Amazon Best-Selling Books Insights Dashboard**
